# LinkedIn Feed Blocker

Chrome extension that removes suggested posts from the LinkedIn news feed. Filters dynamic LinkedIn feed content in real time by traversing nested DOM structures and handling asynchronous updates using the `MutationObserver`.

<img src='https://res.cloudinary.com/ducqdbpaw/image/upload/v1777438517/Group_17_i4hafz.png' height='200'/>

### User Guide

Each switch filters posts based on post type. Toggle the switch to remove the post.
(!Note: Hiding posts means removing their DOM tree element, posts can't be readded to the site's DOM tree. Refreshing the page is required.)
